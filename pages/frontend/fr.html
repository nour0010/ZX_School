دليل شامل لمسار تعلم الـ Front-End من الصفر إلى الاحتراف

ما هو الـ Frontend؟
الـ Frontend، أو ما يُعرف بـ الواجهة الأمامية، هو الجزء الظاهر من الموقع أو التطبيق الذي يتفاعل معه المستخدم مباشرة. يشمل كل شيء يمكن رؤيته والتفاعل معه، مثل الأزرار، القوائم، الألوان، الصور، والنصوص. يتعلق الـ Frontend بكيفية عرض البيانات والتفاعل مع المستخدم بطريقة سلسة وسريعة، ويعتمد على

1. HTML (HyperText Markup Language)

HTML هي لغة ترميز تُستخدم لإنشاء هيكل صفحات الويب. تحدد العناصر الأساسية مثل العناوين، الفقرات، الصور، الأزرار، والنماذج.

أهميتها:
تحدد أماكن العناصر في الصفحة.
تعمل كأساس يتم بناء التصميم عليه باستخدام CSS.

2. CSS (Cascading Style Sheets)

CSS هي لغة تنسيق تُستخدم لتجميل وتنسيق صفحات الويب من خلال تغيير الألوان، الخطوط، الأحجام، الخلفيات، وتوزيع العناصر.

أهميتها:
تجعل الموقع أكثر جاذبية وسهل الاستخدام.
تتيح إنشاء تصميم متجاوب يعمل على جميع الأجهزة.

أُطر العمل (Frameworks) في CSS

أُطر العمل في CSS تساعدك على تصميم المواقع بسرعة وسهولة دون الحاجة إلى كتابة أكواد CSS من الصفر.

اليك اشهرها:
1. Bootstrap (الأشهر والأكثر استخدامًا)
او
2. Tailwind CSS (الأكثر مرونة)

3. JavaScript (JS)

JavaScript هي لغة برمجة تُستخدم لإضافة التفاعل والوظائف الديناميكية إلى صفحات الويب، مثل النقر على الأزرار، إنشاء القوائم المنسدلة، وعرض التنبيهات.

أهميتها:
تجعل الموقع أكثر تفاعلاً وسلاسة.
تُستخدم لإنشاء تطبيقات ويب متقدمة عبر مكتبات مثل React, Vue, و Angular.

أطر العمل (Frameworks) في JavaScript

أطر عمل للواجهة الأمامية (Frontend Frameworks) تُستخدم لتطوير واجهات المستخدم والتعامل مع العناصر الظاهرة في المتصفح.

اليك اشهرها:
React.js: مكتبة مشهورة من تطوير Facebook، تُستخدم لإنشاء واجهات ديناميكية تعتمد على المكونات (Components).

تعلم الAPI في الjavascript

API هو اختصار لـ Application Programming Interface، وهو عبارة عن واجهة تُمكّن البرامج والتطبيقات من التواصل مع بعضها البعض.

في البرمجة، هذا يعني أنك عندما تطلب بيانات من سيرفر (مثل جلب معلومات الطقس)، الـ API هو الذي يتولى إرسال الطلب وجلب البيانات لك.
