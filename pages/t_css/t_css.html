ما هي لغة CSS؟
CSS (Cascading Style Sheets) هي لغة تنسيق تُستخدم لتحديد شكل وتصميم صفحات الويب المكتوبة بـ HTML. تساعد CSS في جعل المواقع أكثر جاذبية وتنظيمًا من خلال التحكم في الألوان، الخطوط، الأحجام، التباعد، التخطيط، والتحريك.
